VS CODE

Visual Studio Code, commonly referred to as VS Code, is an integrated development environment developed by Microsoft for Windows, Linux, macOS and web browsers.Visual Studio Code is proprietary software released under the "Microsoft Software License", but based on the MIT licensed program named "Visual Studio Code — Open Source" (also known as "Code — OSS"), also created by Microsoft and available through GitHub.

Features of VS CODE:

Features include:
support for debugging
syntax highlighting
intelligent code completion
snippets
code refactoring
embedded version control with Git
